소개

2019년부터 Java를 주력으로 웹·앱 개발을 시작해, 점차 웹 백엔드로 분야를 좁히며 전문성을 높여온 7년차 개발자입니다. 같은 기간 학업을 병행해 컴퓨터공학 학사를 취득하고, 국민대학교 인공지능전공 석사 과정을 마무리하고 있습니다(2026.02 졸업). Gemini·Grok 등 LLM과 이미지 생성 AI를 데이터 적재·콘텐츠 생성 파이프라인에 직접 연동·운영하는 'AI를 다룰 줄 아는 백엔드 개발자'입니다.

저는 이렇게 일합니다

  • 변화를 두려워하지 않고, 빠른 습득력으로 좋은 결과물을 만듭니다. 새로운 기술을 도입할 때는 서비스에 미칠 영향도를 철저히 분석합니다.
  • 다른 직군이나 메타지식의 차이가 있는 동료와도 명확하고 솔직하게 소통하고, 문제 상황과 처리 계획을 투명하게 공유합니다.
  • 업무의 본질적 필요성을 먼저 고민하고 개선안을 제시해, 더 나은 결과물을 도출하는 데서 보람을 느낍니다.

보유 기술 (능숙한 순)

  • Backend Spring Boot (Java / Kotlin, Spring Security, Spring Data JPA), Express.js / Nest.js (TypeScript)
  • Infra AWS (ECS, RDS, S3, CloudFront, CloudWatch, CloudFormation), Docker / Kubernetes, CI/CD
  • Architecture Domain Driven Design, Hexagonal, Design Patterns, Mockist Tests, RESTful API
  • Database PostgreSQL, MySQL, Redis
